# draftlab desired features

1. clicking twice to add a wall
2. mouse down and drag to create a wall
3. creating rooms (segmented as 4 walls)
4. holding shift while creating a wall to snap it along a 15 degree axis wheel
5. holding shift while creating a room to make it a perfect square
6. snapping logic of the walls along their nodes. such as snapping nodes to lines, snapping nodes to nodes.
7. fixture library so you can easily drop parametric fixtures such as doors, beds, staircases, etc 
8. ability to display measurements while selecting a wall and changing the measurements to define a new length, thickness, height 
9. 3D mode eventually with ability to change light angle and apply textures to materials
10. parametrically define the length of a wall on the outside and inside. 
11. guide lines along important nearby axises of lines or other important things.
12. mitering the connections of walls at a single node/point
13. ability to save multiple projects within one codec
14. undo/redo certain actions (creating a wall, resizing a wall, moving a wall, changing a walls node positions, etc)
15. etc
